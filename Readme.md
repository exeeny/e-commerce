# E-commerce project

This project is a functional e-commerce application designed to facilitate online shoping experiences. 
Built with Laravel ver. 12 + Inertia.js + React(typescript), using starter pack for auth system and some ui features. Project also includes essential features such as product management, cart system, and order processing.

## Tech Stack:
Frontend: React (TypeScript) + Inertia.js
Backend: PHP Laravel.
UI: Tailwind CSS, shadcn/ui
